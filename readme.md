# Workout Tracker App

A full-stack CRUD application built with ExpressJS, MongoDB Atlas, EJS, and Bootstrap. Users can log, view, and delete workout sessions.

## Features
- Create, read, and delete workout entries
- Responsive Bootstrap layout
- MongoDB Atlas backend
- Secure `.env` configuration
- Deployed on Render

## Technologies Used
- Node.js
- ExpressJS
- MongoDB Atlas
- Mongoose
- EJS
- Bootstrap
